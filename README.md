# Customer Churn Analysis

## Overview
This project analyzes customer churn using SQL and builds a churn prediction model using Python.

## Tools Used
- MySQL
- Python (Pandas, Matplotlib, Scikit-learn)
- Jupyter Notebook

## SQL Analysis
- Churn percentage
- Contract-based churn analysis
- Tenure vs churn
- Monthly charges vs churn
- Payment method impact

## Machine Learning
- Logistic Regression
- Selected features: tenure, MonthlyCharges, TotalCharges, Contract, PaymentMethod
- Model accuracy: ~80%

## Visualizations
- Churn distribution
- Contract vs churn
- Charges vs churn
- Tenure vs churn

## Conclusion
Customer churn is strongly influenced by contract type, pricing, and tenure.
